## 🌟 Vue d'ensemble

**AtmoLab** est une plateforme éducative complète dédiée à l'apprentissage de la météorologie. Elle combine des outils scientifiques avancés, des données en temps réel et des contenus pédagogiques pour offrir une expérience d'apprentissage unique et interactive.

### 🎯 Mission
Rendre la météorologie accessible à tous en proposant des outils pratiques, des simulations interactives et des explications claires sur les phénomènes atmosphériques.

### 🏆 Projet de Maturité (TIP)
Ce projet a été développé dans le cadre d'un Travail de Maturité en informatique, démontrant l'application pratique des sciences météorologiques à travers des technologies web modernes.

## 🚀 Démarrage Rapide

### Prérequis
- **Navigateur web moderne** (Chrome, Firefox, Safari, Edge)
- **Connexion internet** pour les données météo en temps réel
- **(Optionnel)** Clé API OpenWeatherMap pour les données météo réelles

### Installation

1. **Cloner le projet**
   ```bash
   git clone https://github.com/votre-username/TIP2526_AtmoLab.git
   cd TIP2526_AtmoLab
   ```

2. **Configuration API (Optionnelle)**
   ```javascript
   // Dans assets/js/weather-api.js
   const API_KEY = 'votre-cle-api-openweather';
   ```

3. **Lancer l'application**
   - **Page d'accueil** : Ouvrez `index.html`
   - **Station météo** : Ouvrez `home.html`
   - **Simulateur** : Ouvrez `lab.html`
   - **Explications** : Ouvrez `explication-introduction.html`

## 🆘 Support et FAQ

### ❓ Questions Fréquentes

**Q: L'API météo ne fonctionne pas, que faire ?**
R: Vérifiez votre clé API dans `assets/js/weather-api.js` et votre connexion internet.

**Q: Une ville suisse n'est pas trouvée ?**
R: Essayez avec des variantes d'orthographe ou contactez-nous pour l'ajouter.

**Q: Le simulateur ne calcule pas correctement ?**
R: Assurez-vous que tous les paramètres sont dans les plages valides.

**Q: Les graphiques ne s'affichent pas ?**
R: Vérifiez que JavaScript est activé et que Chart.js se charge correctement.
